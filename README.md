# 📊 Daikibo IoT Machine Health Dashboard

## 📌 Project Overview
This project analyzes IoT machine health data from multiple factories using Power BI. The objective is to monitor machine performance, identify unhealthy machines, analyze temperature trends, and provide actionable business insights for preventive maintenance.

---

## 🎯 Business Problem
Manufacturing organizations need to monitor thousands of IoT-enabled machines to minimize downtime and improve operational efficiency.

This dashboard helps stakeholders:
- Monitor machine health across factories
- Identify unhealthy machines quickly
- Track factory and section performance
- Analyze device-type failures
- Support preventive maintenance decisions

## 💡 Key Insights
- Analyzed **160.7K** machine records.
- **99.94%** machines are healthy.
- Only **103** machines require attention.
- **Daikibo-Berlin** has the highest healthy machine count.
- **Section-3** manages the largest machine workload.
- **Laser Welder** and **Laser Cutter** contribute to **88%+** of unhealthy machines.
- Average machine temperature remains stable at **24.99°C**.

---

## ✅ Recommendations
- Prioritize maintenance for Laser Welder and Laser Cutter devices.
- Schedule preventive maintenance for Section-3.
- Continue monitoring average operating temperature.
- Implement automated alerts for unhealthy machines.
- Track failure trends regularly to reduce downtime.
